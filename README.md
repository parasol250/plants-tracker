План развертывания приложения.
1) Скопируйте репозиторий на ПК.
2) Создайте базы данных в MySQL Workbench и Microsoft SQL Server по соответствующим скриптам «plant_tracker_create_DB_MySQL.sql» и «plant_tracker_create_DB_SQL_server.sql» в соответствующих СУБД.
3) (По желанию) Заполните созданные БД тестовыми данными. Скрипты «plant_tracker_insert_MySQL.sql» и «plant_tracker_insert_MySQL.sql» помогут Вам в этом.
4) Откройте проект в среде Eclipse.
5) Добавьте в проект необходимые зависимости:
   ПКМ -> Build Path -> Configure Build Path
в ModulePath добавляете драйверы JDBC для MySQL и SQL Server:
   Add External JARs -> выбираете скачанные jar-файлы драйверов (один для MySQL, второй для SQL Server).
*файл-библиотека «mssql-jdbc_auth-13.4.0.x64.dll» необходим для аутентификации Windows в SQL Server/
7) Устанавливаете плагин JavaFX для Eclipse.
8) Добавляете зависимости для корректной работы JavaFX (распакуйте "openjfx-21.0.2_windows-x64_bin-sdk.zip" и создайте библиотеку (User Library) в Eclipse, в которую добавьте все jar-файлы папки lib; далее через Congigure Build Path добавляете в проект созданную библиотеку).
9) Если проект установлен корректно, то запустите его через класс Main.java. Если проект не запускаете - проверьте корректной выполнение пунктов выше; если все соблюдено - попытайтесь решить проблему с помощью Интернета или свяжитесь с владельцем репозитория.
10) Проведите тестирование приложения
